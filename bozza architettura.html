Announcement: "Suggerimento: se volete centrare…"

Claudio Capobianco
Created Jan 27Jan 27 – Deleted
This post is visible to all teachers in this class. It will be permanently deleted in 3 days.
Suggerimento: se volete centrare qualcosa verticalmente, la cosa più semplice è mettere un'altezza fissa al contenitore (con la proprietà height) e quindi un padding-top all'oggetto che volete centrare, con il numero corretto di pixel. Non è molto responsive ma per ora va bene. In allegato un esempio, su un file modificato a Giulia. Altro suggerimento: almeno in fase di sviluppo, può essere utile mettere un bordo all'elemento contenitore, per vedere quanto spazio occupa. Per ricordarvi che è una cosa di prova potete mettere anche qualcos tipo `border: 3px solid green;`, in modo da avere un margine spesso verde.

Per chi di voi ha problemi a lavorare online o preferisce lavorare in locale, consiglio di usare i seguenti strumenti: - VS Code - l'estensione di VS Code per draw.io (vedi link allegato) - l'estensione di VS Code per Live Server (vedi link allegato) - se VS Code non trova git, probabilmente siete su Windows, lo potete installare con da git-scm Vi allego tutti i link. In questo modo, il flusso di lavoro è: - creo il repository su GitHub e lo inizializzo con un README.md (per questa parte è necessaria la connessione) - copio il link del repository creato - apro VS Code - vado su Version Control, a sinistra (vedi screenshot) - faccio Clone Repository - incollo il link al repository - clono il repository in una cartella locale Se avete installato le estensioni, potete direttamente creare un file con estensione .drawio e modificarlo, creare i file html ed aprirlo su un server locale premendo con il tasto destro sul file e selezionando "Open with Live Server"

USER JOURNEY 1

Santiago Brosio, per motivi lavorativi, si reca nella località di Sanremo la

mattina. La prima cosa che fa è recarsi all'info point più vicino. Qui

viene informato di un servizio in cui, noleggiando un tablet, è possibile

visitare i vari punti di interesse di Sanremo facilmente. Una volta deciso

di usufruire del servizio, Santiago si reca alla Concattedrale di Sanremo

percorrendo l'itinerario che l'app nel tablet gli suggerisce. Giungendo a

destinazione scatta varie foto per il suo lavoro. Consultando ancora

l'app, Santiago vorrebbe recarsi a Villa Nobel notando però che i posti

disponibili sono terminati. Per questo, decide di recarsi al Museo Civico

di Sanremo. Al termine della giornata Santiago torna all'info point

restituendo il tablet.

Page 7 of 7

USER JOURNEY 2

Rebecca De Metri decide di andare in vacanza a Sanremo data la sua

passione per la storia delle città. Durante il viaggio, curiosando su internet,

Rebecca viene a conoscenza di un servizio concesso dall'info point di

Sanremo che le permette, attraverso un tablet noleggiato, di trovare e visitare

i vari punti di interesse in maniera semplice. Appena arrivata a Sanremo,

Rebecca si reca immediatamente all'info point per prendere il tablet. Dopo

pranzo, Rebecca decide di vistare la Chiesa Russa. All'interno della chiesa

però, il tablet non ha segnale. Per questo, usando gli itinerari scaricati in

precedenza, Rebecca può comunque recarsi al Santuario Madonna della

Costa, anche se non sa se ci sono posti disponibili. Al termine della giornata,

Rebecca torna all'info point, restituisce il tablet e va in albergo.
